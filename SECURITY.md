# Security Policy

## Reporting a vulnerability

If you've found a security vulnerability in LGI.tools, please report it
privately — **don't open a public issue**, since that discloses the problem
before it can be fixed.

Email **lgi.tools@pm.me** with:

- a description of the issue and where it lives,
- steps to reproduce or a proof of concept, and
- the potential impact as you see it.

We'll acknowledge your report, investigate, and keep you posted on the fix.
Please give us a reasonable window to address the issue before any public
disclosure.

## Scope

LGI.tools handles EVE Online SSO and stores per-character API tokens, so
authentication, token handling, and access-control issues are especially
valuable. The live site is [https://lgi.tools](https://lgi.tools).

Thank you for helping keep the platform and its players safe.
